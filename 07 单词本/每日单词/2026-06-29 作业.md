

1. 虽然他经历了许多困苦，但他从未放弃学习。
我的：although he exprement many hardship, he never give up studding.
答案：Although he went through many hardships, he never gave up studying.


2. 面对困难时，务实的态度可以帮助我们保持冷静。
我的：face hardship，good attitute can help we to keep clam.
答案：When we face hardship, a practical attitude can help us stay calm.

3. 这篇文章讲述了一个家庭如何克服经济困难。
我的：this page describe a family how to overcome economic hardship.
答案：he article tells how a family overcame economic hardship.




| 错写❌       | 正确用法        | 备注                         |     |
| --------- | ----------- | -------------------------- | --- |
| exprement | experienced | （拼写错；且词义完全不匹配）             |     |
| hardship  | hardships   | （可数名词，many 后用复数）           |     |
| studding  | studying    | （拼写错误）                     |     |
| give      | gives /gave | （时态主谓不一致）                  |     |
| face      | facing      |                            |     |
| attitute  | attitute    | （拼写错误）                     |     |
| clam      | calm        | 形近词拼写错误，clam 是蛤蜊           |     |
| we        | us          | 后面宾语用宾格 us，不用主格 we         |     |
| page      | article     | page = 纸张页面；文章是 article    |     |
| describe  | describes   | - 主语 this article 三单，谓语变形） |     |
| hardship  | hardships   |                            |     |
