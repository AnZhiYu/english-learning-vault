# 高中单词学习进度

## 标记规则

| 状态 | 含义 |
| --- | --- |
| new | 未学习，默认状态 |
| learning | 已学习，但不稳定 |
| review | 已进入复习队列 |
| difficult | 考核出错或使用不熟 |
| mastered | 多轮复习通过 |

## 每日选词规则

- 每天 5 个新词。
- 分类数量根据当天选词自然安排，可以包含核心名词、核心动词、其他。
- 如果当天只学习 5 个词，不强制凑满名词、动词、其他比例。
- 优先从原始词库中未学习的单词挑选。
- 同一天尽量保持主题多样，不把相似词全部挤在一天。

## 每日生成规则

- 先生成“今日新词总览”，把所有新词整合在一张表中展示，并用“分类”列标记核心名词、核心动词、其他。
- 今日新词总览中“词性”和“意思”合并为“词性和意思”一列。
- 再逐个讲解每个新词，包含音标、发音技巧、中国人最容易读错的地方、词性和意思、常见用法、常见搭配、地道短语、近义词、反义词。
- 每个新词生成 3 个例句。
- 每个新词的 3 个中文翻译放在 3 个英文例句上面。
- 例句优先使用：
  - 这个单词的常见搭配。
  - 真实场景和高频表达，避免生硬造句。
  - 之前已经学习过的单词。
  - 图片单词、翻译练习、薄弱点归档中出现过的错题词。
  - 常用连接词，如 because、although、however、also、for example、in fact、as soon as、so。
  - 重点语法，如主谓一致、时态、宾语从句、定语从句、非谓语、被动语态、情态动词、比较级、状语从句、介词搭配。
- 例句知识点要尽量多样，不要 3 句都练同一种语法。
- 关联旧词、重点语法、常用连接词和句子知识点是造句规则，不在每个单词卡片中展示。
- 今日涉及的句子知识点统一合并去重，放在“今日考核”上方的“今日知识点汇总”中。
- 每日单词页不再生成听、说、读、写打卡项，也不再生成掌握度选项。

## 每日考核规则

- 大题一：给出 10 个中文句子，让学习者翻译成英文。
- 10 个中文句子可以同时包含多个今日新词、复习词、错题词和常见搭配。
- 大题二：从今日新词讲解中的例句翻译里随机抽 5 个中文句子，只展示中文，不展示英文，让学习者根据中文自己写英文句子。
- 大题三：给一个与今日新词和复习词相关的主题，写 50-100 个英文单词的小作文。
- 薄弱点归档记录错误造句、翻译错误、搭配错误和错因，用于后续复习安排。
